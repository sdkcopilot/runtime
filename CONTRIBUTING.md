# Contributing

Issues and pull requests are welcome for runtime behavior, API ergonomics, transport features, and public contract improvements.

This repository contains the public, source-available runtime packages for SDKCopilot. The SDKCopilot generator itself is proprietary and is maintained separately.
SDKCopilot is a product of iNAV Canada Corp.

Good issues include:

- incorrect request execution behavior
- auth/query/header/cookie handling problems
- result or error typing issues in the shared runtime contract
- extension points needed by generated SDK users

Generator implementation details may live outside this repository, but feedback about generated SDK behavior is still useful here when it points to runtime contract gaps.
